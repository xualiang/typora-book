### 批量停止容器

```bash
docker stop $(docker ps -a | grep "Exited" | awk '{print $1 }')
```

## 批量删除容器

```bash
docker rm $(docker ps -a | grep "Exited" | awk '{print $1 }') 

或

docker rm -v $(docker ps -aq -f status=exited)
```

### 批量删除镜像 直接使用这一句测试可行

```shell
docker rmi $(docker images | grep "none" | awk '{print $3}')
```
