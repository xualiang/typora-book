## 本地端口转发

```sh
ssh -N -f -L 10.10.71.100:5001:10.10.50.204:5000 root@10.10.50.204
## 在10.10.71.100主机上执行以上命令，命令的效果：访问10.10.71.100的5001端口，相当于访问10.10.50.204的5001端口
```

-L:  代表本地端口转发，监听本地端口，即：监听10.10.71.100:5001（即：执行命令的主机）

10.10.71.100:5001：本地IP和端口

10.10.50.204:5000：转发到的目标IP和端口

root@10.10.50.204：登录到目标主机的root用户

-N:  不打开远程主机shell，最好配合-f使用

-f:  放到后台，否则，如果只使用-N，输入目标主机的密码后，窗口会驻留在那里，一旦退出窗口，隧道即消失
